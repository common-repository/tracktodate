TrackToDate plugin Tracks your package Automatically and notify the store admin and the end customer by means of mail or message.

== Description ==

TrackToDate Plugin Allows you to sync your order from plugin itself to the tracktodate APP where the tracking process is carried out.

Major features in Tracktodate include:

* Automatically checks tracking status for the tracking numbers entered.
* Email and SMS tracking information to customer
* Live Tracking Status in Backoffice "Order View Page" and Customer's "MyAccount Page".
* Create a sharable Tracking information Link and share it to anywhere.

PS: You can use Tracktodate for Free until the free balance is over.

== Installation ==

Upload the TrackToDate Plugin. Install and Activate it and configure.

You're done!
